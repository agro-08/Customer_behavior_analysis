

# Customer_behavior_analysis
Customer Behavior Analysis using Python, PostgreSQL, and Power BI. Performed EDA, SQL analysis, and built dashboards to visualize sales trends, customer insights, and retention data.

## Overview
This project demonstrates an end-to-end Data Analytics workflow using Python, SQL, and Power BI. The objective of the project is to analyze raw business data, clean and transform it, extract meaningful insights, and present the findings through an interactive dashboard and professional report.

The project includes:
- Data loading and preprocessing using Python
- Exploratory Data Analysis (EDA)
- Data cleaning and transformation
- SQL analysis using PostgreSQL / MySQL / SQL Server
- Interactive dashboard creation in Power BI
- Report and presentation preparation using Gamma
